# Code of Conduct

## Our Pledge

We are committed to providing a professional and respectful environment for everyone involved with this project.

## Our Standards

### Positive Behavior

- Professional and respectful communication
- Constructive feedback
- Focus on project goals
- Respect for different viewpoints

### Unacceptable Behavior

- Harassment or discrimination
- Personal attacks
- Trolling or inflammatory comments
- Unprofessional conduct

## Scope

This Code of Conduct applies to all project spaces and public spaces when representing the project.

## Enforcement

Violations can be reported to project maintainers. All complaints will be reviewed and investigated.

### Consequences

- **Warning**: First offense
- **Temporary ban**: Repeat offenses
- **Permanent ban**: Serious violations

## Attribution

This Code of Conduct is adapted from industry best practices.
