# Contributing to Supreme System V5

**⚠️ IMPORTANT: This is proprietary software.**

## For Community Members

Thank you for your interest! While Supreme System V5 is **not open-source**, we welcome:

✅ **Bug reports** (GitHub Issues)  
✅ **Feature suggestions** (GitHub Discussions)  
✅ **Documentation improvements** (with approval)  
❌ Code contributions (proprietary codebase)

## For License Holders

Commercial license holders have access to:

- Private repository for contributions
- Priority feature requests
- Direct engineering support
- Custom integration assistance

Contact: thanhmuefatty07@gmail.com

## Reporting Bugs

1. Check existing issues
2. Use bug report template
3. Include system info
4. Provide reproduction steps

## Feature Requests

1. Open GitHub Discussion
2. Describe use case
3. Explain expected behavior
4. Note if you're a license holder (priority)

## Code of Conduct

Be respectful, professional, and constructive.
