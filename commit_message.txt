## Technical Details

### Algorithm

Monitors validation loss after each epoch. If no improvement for `patience`
consecutive epochs, stops training and restores best model weights.

### Parameters

- patience: Number of epochs to wait (default: 10)
- min_delta: Minimum improvement threshold (default: 1e-4)
- restore_best_weights: Restore best model (default: True)
- verbose: Print messages (default: True)

### Features

- Automatic best epoch detection
- Best model weights restoration
- Lazy PyTorch import (Windows compatibility)
- Comprehensive logging
- Clean API design
