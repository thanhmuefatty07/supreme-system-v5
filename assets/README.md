This directory contains visual assets for Supreme System V5.

## Structure

```
assets/
├── logo.png              # 512x512px logo (to be added)
├── banner.png            # 1280x640px banner (to be added)
├── demo.gif              # Demo video GIF (to be added)
├── architecture.svg      # Architecture diagram (to be added)
├── social-card.png       # 1200x630px social sharing card (to be added)
└── screenshots/
    ├── dashboard.png     # Dashboard screenshot (to be added)
    ├── metrics.png       # Metrics screenshot (to be added)
    └── backtest.png      # Backtest screenshot (to be added)
```

## Usage

These assets are referenced in:
- README.md
- Landing page (index.html)
- Documentation
- Social media posts

## Creation Guidelines

- **Logo**: 512x512px PNG with transparent background
- **Banner**: 1280x640px PNG for README header
- **Demo GIF**: Max 10MB, optimized for GitHub
- **Architecture**: SVG format for scalability
- **Social Card**: 1200x630px PNG for sharing

## Tools

- Logo/Banner: Canva, Figma, or Fiverr ($50-150)
- Demo GIF: LICEcap (free) or OBS Studio
- Architecture: Draw.io or Excalidraw
- Screenshots: Standard screenshot tools
